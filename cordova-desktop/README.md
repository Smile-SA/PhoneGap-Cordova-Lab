#Cordova Desktop

A project aiming at mocking PhoneGap JavaScript API to run a PhoneGap App in a desktop browser.

## Features

Simply replace the Cordova JS script by this one when you want to run your app in a desktop browser.

## Requirements

jQuery is needed.
